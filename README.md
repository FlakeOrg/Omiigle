# Omiigle
An app for the 3DS similar to Omegle.
## What is Omiigle?
Omiigle is like Omegle. But it's for the 3DS. Instead of using webcams and voice messages, we use Miis and text messages. You can use a variety of features, like:
* Local connection
* Online connection
* Sharing friend codes between systems
* Cross-platform (you can chat with Wii U users!)
* Sharing Miis
* Using Miis
* Reporting system
